# 2026-03-02 (Day 4) - 모듈, 패키지 & 타입 힌트

### 오늘 한 것
- 모듈과 패키지 개념 학습
- import 방식 4가지 비교 (import X, from X import Y, 여러 개, 별명)
- `__init__.py`의 역할과 편의 등록
- `if __name__ == "__main__":` 패턴
- 타입 힌트 기초 (기본 타입, Optional, Union)
- 실습: equipment 패키지 만들기 (sensor, alarm 모듈)
- 연습 문제: pressure 기능 추가 (sensor + alarm + main + __init__.py 전부 수정)

### 배운 것
- `__init__.py`는 폴더를 패키지로 만드는 "간판" 역할
- `.sensor`의 `.`(점)은 "같은 패키지 안에서" (상대 임포트)
- 모듈에 함수 추가하면 `__init__.py`에도 등록해야 패키지 레벨에서 사용 가능
- `from X import *`는 이름 충돌 위험 → 실무에서 쓰지 말 것
- 타입 힌트는 강제가 아닌 힌트일 뿐, `mypy`로 별도 검사
- `__name__`은 직접 실행 시 `"__main__"`, import 시 모듈명
