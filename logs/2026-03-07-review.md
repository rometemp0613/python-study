### 오늘 한 것
- 기초 문법 ~ NumPy까지 전체 복습 (퀴즈 + 코드 챌린지)
- 5라운드: 개념 퀴즈 → 코드 챌린지 → 개념 퀴즈 → NumPy 챌린지 → 종합 클래스 챌린지

### 잘한 것
- 로직/흐름 설계는 항상 정확
- enumerate, 기본값 인자, zip, @property, finally, set 등 개념은 잘 기억

### 자주 헷갈리는 패턴 3가지
1. **self 빠뜨리기** — 메서드 정의, 인스턴스 변수 접근 시 항상 `self.`
2. **다른 언어 문법 섞임** — `${}` (JS의 f-string), `catch` (Java) → Python은 `{}`, `except`
3. **NumPy 활용** — 반복문 대신 axis, 불린 인덱싱 한 줄로 쓰는 연습 필요

### 추가 발견
- lambda key 활용 (`sorted(data, key=lambda x: x[1])`) 아직 불확실
- dict 순회 시 `.items()` 빠뜨림
- `__init__` 대신 클래스명으로 생성자 정의하려 함
- `avg()` 함수는 Python에 없음 → `sum()/len()` 또는 `np.mean()`
