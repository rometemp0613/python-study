# 2026-03-03 (Day 5) - NumPy 기초

### 오늘 한 것
- ndarray 개념 및 리스트와의 차이 학습
- 배열 생성 함수 (zeros, ones, arange, linspace, random)
- 인덱싱 & 슬라이싱 (1차원, 2차원)
- 불린 인덱싱으로 조건 필터링
- 벡터 연산 (반복문 없는 배열 연산)
- 브로드캐스팅 개념
- axis 개념 (axis=0 세로, axis=1 가로)
- 연습 문제 3개 + 퀴즈 3개 통과

### 배운 것
- ndarray는 같은 타입만 저장, 메모리 연속 배치 → 빠름
- 불린 인덱싱: `arr[arr > 5]` — 배열 이름 그대로 사용
- 조건 조합 시 `&`, `|` 사용 (`and`, `or` 아님), 괄호 필수
- `np.arange`는 끝값 미포함 (range와 동일)
- `axis=0`은 세로 방향(행 따라), `axis=1`은 가로 방향(열 따라)
- `keepdims=True`로 브로드캐스팅 시 차원 유지
