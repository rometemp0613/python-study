# 2026-02-28 학습 일지

## 학습 주제
- 02. Python 내장 테스트: unittest & doctest

## 배운 내용
- unittest.TestCase: Python 내장 테스트 프레임워크, xUnit 스타일
- assert 메서드들: assertEqual, assertAlmostEqual, assertRaises 등
- setUp/tearDown: 테스트 전후 초기화/정리 (pytest fixture와 유사)
- doctest: docstring 안에 >>> 입력 + 기대 출력으로 테스트 겸 문서화
- doctest 한계: 에러 테스트 어려움, 복잡한 셋업 부적합, 출력 형식 민감
- pytest와의 비교: pytest가 더 간결하고, unittest 테스트를 그대로 실행 가능

## 실습
- unittest.TestCase 기반 테스트 클래스 작성
- doctest 문법으로 함수 문서화 + 테스트
- pytest.approx(), pytest.raises()와 비교

## 비고
